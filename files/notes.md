# NOTES

## STATIC ARRAY

CLASS:
    PROPERTIES:
        SIZE INT
        DATA INTEGERS ARRAY OF SIZE SIZE

    METHODS:
        INSERT(INDEX) INSERTS VALUE AT INDEX
        ACCESS(INDEX) CHANGES VALUE AT INDEX
        UPDATE(INDEX) CHANGES VALUE AT INDEX
        DELETE(ELEMENT) DELETES FIRST OCCURENCE OF ELEMENT AND RETURNS NEW ARRAY OF SIZE - 1
        SUM RETURNS SUM OF ALL ELEMENTS
        REVERSE RETURNS ARRAY IN REVERSE ORDER OF ELEMENTS
        SLICE(START, END, SKIP) RETURNS NEW ARRAY FROM SLICING OPERATION
        MERGE SORT() RETURNS SORTED ARRAY
        QUICK SORT(PIVOT) RETURNS SORTED ARRAY
        LINEAR SEARCH(VALUE) RETURN INDEX OF VALUE IN ARRAY OR FALSE IF ABSENT
        BINARY SEARCH(VALUE) RETURN INDEX OF VALUE IN ARRAY OR FALSE IF ABSENT
        MAX() RETURN MAX VALUE IN ARRAY
        MIN() RETURN MIN VALUE IN ARRAY
        COUNT(VALUE) RETURN NUM OF OCCURENCES OF VALUE IN ARRAY
        ROTATE(STEPS, DIRECTION) RETURN ARRAY AFTER MOVE ALL ELEMENTS STEPS IN DIRECTION WRAPPING AROUND SIZE, STEPS = STEPS % SIZE
